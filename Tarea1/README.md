# Descripción de la app

Esta aplicación tiene como finalidad demostrar el uso de los componentes básicos de la interfaz de usuario en Android. Para este fin se hizo uso de Activities y Fragments.

Se utilizó una Activity principal y 5 Fragments, en los cuales se muestra un elemento de de interfaz diferente.

La aplicación se divide en 5 secciones principales, cada una dedicada a un grupo de elementos de UI diferente:

## TextFields (EditText):
  Explica cómo capturar entradas de texto del usuario.
  Permite escribir y visualizar el texto ingresado en tiempo real.
## Botones:
  Muestra la diferencia entre botones con texto y botones con imágenes.
  Incluye eventos de clic que actualizan el estado en pantalla.
## Elementos de Selección:
  Demuestra el uso de CheckBox, RadioButton y MaterialSwitch.
  Se puede ver el estado actual de cada selección (activado/desactivado).
## Listas:
  Presenta una lista desplazable eficiente cargada con datos de ejemplo.
  Sirve como modelo para mostrar conjuntos de datos repetitivos.
## Elementos de Información:
  Muestra componentes de salida como TextView, ImageView y una ProgressBar.
  Permite interactuar con la barra de progreso para ver su funcionamiento.

# Instrucciones de Uso

## 1. Navegación Principal
* Utiliza la Barra de Navegación Inferior para moverte entre las 5 categorías de la app. Cada icono representa un fragmento diferente.

## 2. Probar los Elementos
* Sección de Textos: Escribe en el campo de entrada y presiona el botón "Mostrar texto" para ver el resultado debajo.
* Sección de Botones: Haz clic en el botón de texto o en el botón de cámara para ver cómo cambia el mensaje de "Estado".
* Sección de Selección: Marca la casilla de términos, selecciona una opción (A o B) o activa el switch de notificaciones. Verás un resumen de tus elecciones en la parte inferior.
* Sección de Listas: Desliza el dedo de arriba hacia abajo para navegar por la lista de elementos generados.
* Sección de Información: Pulsa el botón "Actualizar progreso" para aumentar la barra de carga. Al llegar al final, esta se reiniciará automáticamente.
